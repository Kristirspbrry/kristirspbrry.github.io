A city guide app in multiple languages

This app helps travellers to find their way around cities as a tourist. It's main feature is to guide the user through museums and other sights in a city.

I worked with creating maps a lot on this project. The main focus was making them easy to understand and accessable to the user. One of the features that assists with that in this case is the mobile device showing the user exactly where they are on the map.

We aimed to reduce the amount of features so only the most important remained. We worked with them to amplify the user experience. For example, the search feature has a number of filters already preselected for the user that can be altered, however, that saves the user time in thinking how the search feature works and how to search for things.
